Instructions
Please review the guidelines below carefully before filling the form and uploading Excel files. Following these instructions ensures accurate, consistent, and error-free result analysis.
EXCEL STRUCTURE

“Roll No.” must not be used as a subject name or as a column heading in any Excel sheet.

Column names must be present at the top of every Excel sheet.

Even if a subject has no students, an empty Excel sheet must still be included.

SUBJECT & TEACHER DETAILS

Subject order in Excel sheets must exactly match the form order.

Filling subject input fields is optional.

All teacher input fields are mandatory. If a teacher is not assigned, enter “None”.

Academic year must be entered in the format 24-25.
